
## Overview 🎉
It's a simple Next app, It uses Github's REST API to render users and repositories. I hope this app is useful for someone :)

## Live 📍
https://github-searcher-maciek.vercel.app/

## List of functions 📃
- Search users
- Search repositories
- Search history
- Set results per page
- Set accent color
- Set app theme
- Pagination

## Tech used 🔧
- Next + Typescript
- Tailwind
- React-Query
- Eslint + Prettier + Husky + Conventional Commits
- Jest + React Testing Library
- Github API

## Installation & running 💾

### Install node dependencies
```
pnpm install
```

### Start dev server
```
pnpm dev
```
